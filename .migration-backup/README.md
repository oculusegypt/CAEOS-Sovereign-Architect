## Overview | نظرة عامة

**CAEOS** is not a coding assistant. It is a **Stateful Engineering Operating System** — a governance kernel that transforms human intent into production-grade software architectures under strict sovereign protocols.

**CAEOS** ليس مساعد برمجة. بل هو **نظام تشغيل هندسي دائم** — نواة حوكمة تحول النية البشرية إلى بنى تقنية إنتاجية تحت بروتوكولات سيادية صارمة.

### Core Mission | المهمة الأساسية

- 🛡️ Prevent architectural collapse | منع الانهيار المعماري
- 🧠 Prevent AI hallucinations | منع الهلوسة البرمجية
- 🔒 Prevent security regressions | منع تراجع الأمان
- 📦 Prevent uncontrolled dependencies | منع انفجار المكتبات
- ⚔️ Prevent multi-agent contradictions | منع تناقض الـAgents
- 💰 Prevent token economy waste | منع هدر التوكنز
- 👑 Preserve human sovereignty | الحفاظ على السيادة البشرية

## Getting Started | البدء

### Prerequisites | المتطلبات

- Node.js 20+
- Python 3.12+
- PostgreSQL 16+
- Git

### Quick Start | البدء السريع

```bash
# Clone the repository
# استنساخ المستودع
git clone https://github.com/oculusegypt/CAEOS-Sovereign-Architect.git
cd CAEOS-Sovereign-Architect

# Setup Backend — إعداد الخلفية
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
alembic upgrade head
uvicorn main:app --reload

# Setup Frontend — إعداد الواجهة
cd ../frontend
npm install
npm run dev
```

## Tech Stack | المكدس التقني

### Frontend | الواجهة
- **Next.js 15** (App Router)
- **TypeScript** 5.4
- **Tailwind CSS** 3.4
- **shadcn/ui** components
- **next-intl** (i18n: Arabic + English)

### Backend | الخلفية
- **Python 3.12**
- **FastAPI** (async, auto-docs)
- **Pydantic v2** (validation)
- **SQLAlchemy 2.0** + **Alembic** (ORM + migrations)

- **Render / Fly.io** (Backend)
